# CleaningData
In this repository you can find the required scripts and data to properly analyze the files found in UCI_HAR_DATA.zip to get a clean and tidy data set consisting of the mean and standard deviation of each measurement

# Steps
1. First download the files from UCI_HAR_DATA.zip
2. Then run the file named run_analsis.R. This will create your tidy data set
3. Finally use the write.table in console to write your dataset into a text file
